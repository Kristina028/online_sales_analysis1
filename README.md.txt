# Online Sales Analysis

## Opis projekta
Ovaj projekat služi za upravljanje proizvodima i korpom u online prodavnici koristeći Python i OOP.

## Klase
- **Product**: atributi name, price, quantity; metode za prikaz i ažuriranje.
- **ProductManager**: upravlja listom proizvoda; dodavanje, prikaz, ukupna vrednost, uklanjanje.
- **Cart**: upravlja proizvodima u korpi; dodavanje, prikaz i ukupna vrednost korpe.

## Kako koristiti
1. Pokreni `main.py`.
2. Proizvodi će biti dodati i prikazani.
3. Tri proizvoda će biti dodata u korpu i prikazana ukupna vrednost.
